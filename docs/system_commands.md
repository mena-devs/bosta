# system_commands

System and telemetry commands to monitor and manage the bot remotely.

### Example Usage

```
@bosta uptime

#> Output
info:    Forever processes running
data:        uid  command       script  forever pid  id logfile                 uptime       
data:    [0] ulYK /usr/bin/node main.js 978     3300    /root/.forever/ulYK.log 34:1:2:2.285 
```

```
@bosta recents

#> Output
There you go: ... list of users ...
```
